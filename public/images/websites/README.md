# Website Preview Images

Add your screenshot images here. They are displayed as the static preview before the video plays on hover.

| File | Content |
|------|---------|
| `landing-page.jpg` | Screenshot of your landing page example |
| `ecommerce.jpg` | Screenshot of your e-commerce example |
| `portfolio.jpg` | Screenshot of your portfolio example |

**Recommended specs:**
- Format: JPG or PNG
- Resolution: 1200×800px (3:2 ratio works best with the card layout)
- File size: keep under 500 KB for fast loading

If a file is missing, the card automatically falls back to the placeholder Unsplash image — no crash.

# Website Preview Videos

Place your demo screen-recordings here. The video player uses `autoPlay muted loop playsInline` so files must be MP4 (H.264).

| File | Content |
|------|---------|
| `landing-page.mp4` | Screen-recording of your landing page example being scrolled |
| `ecommerce.mp4` | Screen-recording of your e-commerce example being scrolled |
| `portfolio.mp4` | Screen-recording of your portfolio example being scrolled |

**Recommended specs:**
- Format: MP4 (H.264 + AAC)
- Resolution: 1920×1080 or 1280×720
- Duration: 15–30 seconds, looping naturally
- File size: keep under 15 MB per file for fast loading

If a file is missing the video area shows as black — the page will not crash.
